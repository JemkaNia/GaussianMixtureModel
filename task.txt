Реализовать скрипт, генерирующий случайный массив с заданными параметрами. Скрипт
запрашивает у пользователя следующие данные:
 количество гауссиан,
 среднее, дисперсию для каждой гауссианы,
 количество точек, генерируемых с такими параметрами.
После того, как с помощью numpy.random.normal сгенерированы гауссианы, их необходимо
перемешать функцией numpy.random.shuffle для получения итоговых данных.
Реализовать получение заданных пользователем параметров с помощью метода GMM библиотеки
scikit-learn. Это необходимо для того, чтобы убедиться в правильности работы генератора тестовых
данных и метода.
Реализовать метод GMM самостоятельно на Python, используя EM-алгоритм. Убедиться, что
собственная реализация выдаёт тот же результат, что и библиотечный метод.
Написать функцию, которая принимает на вход отсчёты сигнала и частоту дискретизации и выдаёт
вектор параметров: среднее, дисперсию и амплитуду для каждой из гауссиан смеси. Например,
если количество гауссиан равно трём, то функция должна вернуть вектор из 9 чисел. Функция
должна считать параметры гауссиан по усреднённому спектру. Для этого исходный сигнал
разбивается на блоки по 1024 значения, для которых затем выполняется БПФ (функцией
numpy.fft.rfft) и берётся амплитуда значений (функция numpy.abs). Для каждого блока таким
образом получится массив из 512 элементов. Затем эти массивы усредняются – эти и будет
усреднённый спектр.
По полученным признакам оценить точность классификации с помощью данного кода (см.
информацию в группе).
